# Model F3.3: Parameter definition by equation

## Objective
- Introduce parameter definition specified through an equation.
- This is the same as model CE3.3, but here the quantity $λ_s(t)$ is defined by the force of infection.
- This can be a useful way to avoid unnecessarily long algebraic expressions shown on the model.
- Unlike reparameterisation, defined quantities can depend on time-varying quantities such as populations, and don't use splines.

## Model
- A single population-based species 'People' is created.
- This contains a classification 'DS' which stands for 'disease status' and a classification 'Sex'.
- DS contains three compartments: susceptible $S$, infected $I$ and recovered $R$. Together they are known as the “SIR model”.
- A force of infection $λ_s(t)$ specifies the rate at which individuals become infected.
- This is defined in terms of a transmission rate $β_s$ that determines the rate at which individuals with sex $s$ become infected.
- A recovery rate $γ_s$ determines the rate at which individuals recover with sex $s$.
- $N$ is the total number of individuals (this model assumes frequency dependent transmission).
- Classification 'Sex' contains the compartments $M$ and $F$ for male and female.

## Population
- This consists of 10000 initially susceptible individuals and 2 infected.
- There is a 50/50 split between males and females.
